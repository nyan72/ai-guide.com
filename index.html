Claude Code ガイド

ターミナルから離れずに、AIと対話しながらコーディングする

Claude Code とは？

Anthropicが提供する、コマンドラインインターフェース（CLI）専用のAIエージェントです。コードの編集、テストの実行、git操作などをターミナル上ですべて完結させます。

クイックスタート

1. インストール

Node.js環境で以下のコマンドを実行します。

npm install -g @anthropic-ai/claude-code

2. 初期設定

認証を行うために起動します。

claude

3. 使い方

プロジェクトのルートディレクトリで質問や指示を投げます。

# 例: バグの修正を依頼する
> /fix index.js のバリデーションエラーを直して

主な特徴

  • ファイル操作: 複数のファイルをまたいだ書き換えが可能。
  • コマンド実行: テストを自分で走らせ、エラーが出たら自己修正。
  • Git連携: 修正内容を理解し、適切なコミットメッセージを作成。
